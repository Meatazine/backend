# “肉大师”电子杂志制作工具

“肉大师”是一款基于浏览器的内容制作工具。我们利用HTML5技术，让内容生产者得以使用最常见的工具来制作能够在各类移动设备上观看的内容——初步以电子杂志为主。肉大师编辑器目前只支持Chrome浏览器，输出的杂志可以在iOS系统和Android系统上观看。
